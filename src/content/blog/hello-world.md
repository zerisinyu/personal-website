This is a sample post. The blog works like this:

1. Write a markdown file in `src/content/blog/`.
2. Give it a `title` and a `date` in the frontmatter.
3. Commit and push — the site rebuilds and the post appears here and in the [RSS feed](https://zerisinyu.github.io/personal-website/rss.xml).

Body text is set at a comfortable measure (~66 characters), with a slightly
larger size and relaxed line height for long-form reading.
